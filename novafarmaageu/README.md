# Nova Farma Ageu - v3

Projeto React + Vite + Tailwind + Firebase Firestore.
Versão corrigida: separação por loja, uppercase nome, export CSV/PDF, realtime.

## Como usar localmente
1. Instale dependências:
   npm install
2. Rodar em dev:
   npm run dev

## Deploy na Vercel
1. Push para GitHub.
2. Import project no Vercel.
Build command: npm run build
Output dir: dist
