# Nova Farma Ageu - v2

Projeto React + Vite + Tailwind + Firebase Firestore.
Configuração pronta para deploy em Vercel.

## Como usar localmente
1. Instale dependências:
   npm install
2. Rodar:
   npm run dev

## Deploy na Vercel
1. Push para GitHub.
2. Import project no Vercel.
Build command: npm run build
Output dir: dist
